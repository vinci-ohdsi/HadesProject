Description

Plot(s) showing for the databases selected the number of events (y axis) during a given time period (1, 30 or 365 days) (x axis) for the target and outcome cohorts selected.

Options

It is required to select a target and outcome cohort and then click the generate report button.

Plots will be displayed with the results from multiple data sources and timespans.

The legend to the right of the plots shows the meanings of the colors.

The colors and meanings are below.

  • Coral - first-after last target end
  • Yellow-green - first before first target start
  • Green - first during first
  • Turquoise - subsequent after last target end
  • Blue - subsequent before first target start
  • Pink - subsequent during first
