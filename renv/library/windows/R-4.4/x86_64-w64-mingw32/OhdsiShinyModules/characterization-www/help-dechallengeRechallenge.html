Description

A table will be displayed for the databases and selected cohorts.

Options

It is required to select a target and outcome cohort and then click the generate report button.

A table will be displayed with the results from multiple data sources and timespans.

The columns displayed in the table are described below.

Name Description
Database Name for a given database for the results, e.g. IBM CCAE; the drop down box allows the filtering of results to 1 or more databases.
dechallengeStopInterval  
dechallengeEvaluationWindow  
numExposureEras Distinct number of exposure events in target cohort. 
numPersonsExposed Distinct persons exposed in target cohort. A person must have at least 1 day exposure to be included.
numCases Distinct persons in outcome cohort. A person must have at least 1 day of observation time to be included.
dechallengeAttempt Distinct count of exposures after the outcome. 
dechallengeFail Distinct count of exposures resulting in an outcome after the first outcome. 
dechallengeSuccess Distinct count of exposures after the first outcome without an outcome.
